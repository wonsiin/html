웹을 구성하는 언어의 종류

  1. HTML(Hypertext Markup Language)
    • 웹 페이지를 기술(디스크립션)하기 위한 마크업 언어
    • 웹 페이지의 내용(content)와 구조(structure)를 담당
  2. CSS(Cascading Style Sheet)
    • Presentation를 담당.(웹페이지를 꾸며주는것)
    • HTML문서의 디자인 처리
  3. JS(Javascript) 웹에서 동작하는 스크립트 언어
    • HTML, CSS의 문서 내용에 접근하여 동작(Behavior)를 만드는 언어
    • 프로그래밍 언어
    • 제어문, 반복문 등이 있어 프로그래밍 언어와 같이 동작을함
    • 웹브라우저 내에 포함된 해석기(엔진)가 JS를 해석해서 동작을 시킴
      (Chrome, Firefox, Safari, Edge(-> IE), 0pera) : 5대 Major 브라우저
